Algorithm

  1. Fetch https://crt.sh/?q=%25.thite.site&output=json.
  2. Normalize names to lowercase, remove wildcards, keep thite.site and subdomains.
  3. Probe https://host then http://host for status, title, and description.
  4. Write sorted rows to data/subdomains.json with scan timestamps and source errors.

UI layout

  1. Hero: title, daily-scan status, and last refresh time.
  2. Controls: client-side search and status filter.
  3. Cards: host, title, status, protocol, description, and discovered source.
